# Three.js — Lights Playground

A small learning project to explore different light types in Three.js and control them in real-time using **lil-gui**.

## Demo
- Toggle lights ON/OFF
- Adjust intensity, color, position, and other properties
- Visual helpers to inspect light direction and area
- Responsive canvas and orbit controls

> This project is built for learning / portfolio purposes — great to show grasp on 3D lighting basics.

---

## Tech stack
- Three.js (WebGL)
- lil-gui (light control panel)
- Vanilla JS 
- Vite (recommended dev server)
